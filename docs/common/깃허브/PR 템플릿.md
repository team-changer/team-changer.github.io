### PR Template 위치

repository의 아래 위치에 md 파일을 생성한다.

`.github/PULL_REQUEST_TEMPLATE.md`

### 템플릿

```md
### 📌 관련 이슈

- closed #이슈번호

### 📁 작업 설명

- 설명

### 📸 작업화면(선택)

### 💻 미완료 태스크(선택)

- [ ] Task1
```
